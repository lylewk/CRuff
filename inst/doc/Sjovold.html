Let’s see for the total data set whether the proportion of cases in the 50 percent prediction interval actually contains 50 percent of the data. That is a straight-forward binomial test:

sto=573+466+23
print(sto)
#> [1] 1062
print(binom.test(466,sto,0.5))
#> 
#>  Exact binomial test
#> 
#> data:  466 and sto
#> number of successes = 466, number of trials = 1062, p-value = 7.4e-05
#> alternative hypothesis: true probability of success is not equal to 0.5
#> 95 percent confidence interval:
#>  0.4086835 0.4692460
#> sample estimates:
#> probability of success 
#>              0.4387947

OK, not so good. The probability value is quite low, because the 95% confidence interval for the proportion in the 50% prediction interval is from about 0.4087 to 0.4692. It is even worse if we compare the proportion of cases below the 50% PIs to the total of those above and those below.

sto=573+23
print(sto)
#> [1] 596
print(binom.test(573,sto,0.5))
#> 
#>  Exact binomial test
#> 
#> data:  573 and sto
#> number of successes = 573, number of trials = 596, p-value < 2.2e-16
#> alternative hypothesis: true probability of success is not equal to 0.5
#> 95 percent confidence interval:
#>  0.9426559 0.9753824
#> sample estimates:
#> probability of success 
#>              0.9614094

OK, so that shows that the bulk of the cases (actual statures) are below the 50% PIs. We would have expected about half should be below and half should be above. But, that is not happening because the estimated statures are biased. Generally, the estimates are greater than the actual statures, as in the plot.

Now let’s do the binomial test for the proportion in the 50% prediction intervals for the overall data.

sto=14+22+6
print(sto)
#> [1] 42
print(binom.test(22,sto,0.5))
#> 
#>  Exact binomial test
#> 
#> data:  22 and sto
#> number of successes = 22, number of trials = 42, p-value = 0.8776
#> alternative hypothesis: true probability of success is not equal to 0.5
#> 95 percent confidence interval:
#>  0.3641780 0.6799595
#> sample estimates:
#> probability of success 
#>              0.5238095

Great! The probability value is (quite) high and the 95% confidence interval for the proportion in the 50% PIs contains 0.5. Now let’s look for bias.

sto=14+6
print(sto)
#> [1] 20
print(binom.test(14,sto,0.5))
#> 
#>  Exact binomial test
#> 
#> data:  14 and sto
#> number of successes = 14, number of trials = 20, p-value = 0.1153
#> alternative hypothesis: true probability of success is not equal to 0.5
#> 95 percent confidence interval:
#>  0.4572108 0.8810684
#> sample estimates:
#> probability of success 
#>                    0.7

Also great! The Sjøvold (1990) equation is doing well on a short-statured sample, as we might expect for a method (reduced major axis) that is supposed to correct for reference sample composition (though less so than for classical calibration?)
